# SQL Practice
SQL problems covering the core analytical patterns I use day-to-day as a data analyst.

## What's in here
- **Window functions** - ranking, running totals, LAG/LEAD, moving averages
- **Funnel analysis** - conversion rates, drop-off points, step-by-step breakdown
- **Cohort & retention** - weekly/monthly retention curves, cohort comparison
- **A/B testing** - significance checks, conversion rate comparison, guardrail metrics
- **Product analytics** - DAU/MAU, feature adoption, user segmentation

## Sources
Problems sourced from DataLemur, LeetCode (SQL), and similar platforms

## Stack
SQL (Microsfot SQL Server syntax) — compatible with Snowflake and BigQuery with minor adjustments.
